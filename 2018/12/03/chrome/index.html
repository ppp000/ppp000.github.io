Chrome extensions

Dec 03 2018

The main article is Google Chrome extension, the following extensions are frequently-used and high-performance as a code livestock.

  • CSS Peeper
    Extract CSS and build beautiful styleguides.

  • ESI Stylish
    Stylish is a user style manager that adjusts the appearance of most mainstream web pages.

  • Evernote Web Clipper
    It can save the web content to the Evernote account, for offline and cross-device reading, even search.

  • FPS extension
    It is a tiny tool. This extension shows a current FPS(frames per second) on a web page.

  • Google PageSpeed Insights API Extension
    Get the Google PageSpeed Insights score for all channels

  • Imagus
    Enlarge thumbnails, and show images/videos from links with a mouse hover.
