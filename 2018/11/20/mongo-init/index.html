Mongo init in Window10

Nov 20 2018

Prepare

System：windows10

  • download mongo.msi

Step

Please tick compass-install.exe on the component selection page.

  1. Install Mongo on the C:\mongodb through a custom installation, manually create C:\mongodb\data\db and C:\mongodb\log\mongodb.log.

  2. Runs cmd as a administrator and jumps to the C:\mongodb\bin

  1. Runs a new cmd as a administrator and jumps to the C:\mongodb\bin

  2. Open http://localhost:27017/ in your browser, the following figure shows the normal MongoDB service.

  3. Until this step, you can run and connect MongoDB service, but it’s one-off. We can automatically run the MongoDB service with your Window startup.

  4. Type win + r to runs service.msc, you can see a list of MongoDB service.

Visualization tool: robo3t

open shell

Remarks

Bin is the executable file under the current directory.

MongoDB refers to the service itself.

You can connect MongoDB service through mongo.

In non-relational database such as Mongo:
connection > database > collection > document or BSON document（similar to row of Mysql）> filed（similar to column fo Mysql） > index
